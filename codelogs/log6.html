Code Log 6: Reference Quest

CSS: Media Queries!

What sites were visited?

  • W3Schools
  • Mozilla
  • Mimo
  • CSS-Tricks

What are the differences in the information presented on each site?

W3Schools: Pretty straightforward. It gives a basic definition and a bunch of simple examples, plus some different situations like screen sizes and orientations.

Mozilla (MDN Web Docs): Actually explains how media queries work, breaks down the syntax, and includes more advanced stuff. Feels more like a full reference than a quick guide.

Mimo: Super simple and beginner-focused. It explains the idea quickly, which is good if you’re just trying to understand the basics fast.

CSS-Tricks: More real-world focused. It explains things more conversationally and gives tips on how people actually use media queries in projects.

What features do you think a code reference page should offer?

It should have clear explanations, good examples, and show how things are actually used in real situations. Also helpful if it’s easy to navigate and updated. Interactive examples or demos would make it better.

Do any of your 4 resources stand out as particularly good or bad?

MDN is probably the best overall since it’s super detailed and reliable. CSS-Tricks is also really useful for practical tips. W3Schools is nice for quick help, but it's kind of basic. Mimo is the simplest, so it’s good for beginners,s but not great if you need more detail.
