CSS Color!

What do these CSS properties do?

color

Colors are specified by using a predefined color name, or with color values, like RGB or HEX. The color value can also be altered to make it transparent. Can be used to change the color of backgrounds, text, and borders.

background-color

This property specifies the background color of an element or a page. This can also be used to set the background color for any HTML element, which need to be specified in the code.

border-color

This property specifies the style, width, and color of an element's border. Often used with border-style to make borders cooler.

How does CSS accept color as a value? Provide examples of colors in these styles:

Named

Color names can be used directly as values in CSS, this is a simple way to input color.

Hexadecimal

Also known as hex codes, hexadecimals use the # sign follow by digits that define the red, green, and blue components.

RGB

Specifies color using the red, green, and blue color model.

RGBA

Similar to RGBA, but adds an alpha channel value, "a" to adjust transparency.

HSL

Specifies hue, saturation, and lightness, that is often used by some designers.

HSLA

The same way RGBA is like RGB, HSLA adds the alpha channel for opacity.
