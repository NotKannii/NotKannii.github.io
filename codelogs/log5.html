Code Logs 5: Images

What's the difference?

Images in HTML

Images are primarily used in HTML to embed visual content into a web page using the img tag. Images are not technically inserted into the page but are linked to it, creating a holding space for the referenced image file. Images can be used in several other sophisticated ways to enhance the visual appeal, performance, and accessibility of a website.

Images in CSS

When an image provides semantic value or is a part of the page's content, the HTML img tag is the preferred method for inclusion. CSS is then used to control its appearance and layout. Images that are purely decorative or part of the design/user interface should be added using the CSS background-image property.
