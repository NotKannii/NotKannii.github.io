"What is the difference between absolute and relative?"

What is absolute?

The absolute CSS tag is a unit of length. These lengths are usually fixed in a layout for one type of screen layout. For example, if a website is designed with absolute measurements, it will look normal on a computer, but small and unreadable on a phone.

What about relative?

The relative CSS tag is also a unit of length. However, unlike its absolute counterpart, relative measurements are responsive and adapt to the screen it is being shown on. It provides the best flexibiliy across different platforms and devices, making it readable no matter how big or small a screen is.

"Let's go over some measurements, shall we, zebras."

Absolute Measurements

  • Pixels: px | Also known as "picture element", px is the smallest element on the screen. It is the standard measurement to ensure consistent visuals across different platforms and resolutions, relating directly to a device's display settings.
  • Points: pt | pt is the OG, being a traditional typographical measuring unit back from the printing industry. This unit is mainly used in print-specific formats, where physical and fixed measurements are utilized. This unit does not scale automatically like relative tags do, so content measured in pt might be hard to read if not presented correctly.

Relative Measurements

  • Parent: em | em, short for "em height" is the unit used for creating scalable and responsive designs. em is relative to the font-size of the nearest parent or current element, depending on where it is being used.
  • Root: rem | rem, also called "root em" or "root ephemeral unit" is the unit used for creating scalable and responsive designs. Unlike the em unit, rem is relative to the font-size of the root element on the HTML document. This means that if a user has adjusted their browser's default font size, the page will still be shown correctly.
  • Viewport Width: vw | vw, or "viewport width", is a unit for scalable and responsive designs. However, it references the size of the user's browser window. 1vw is equal to 1% of the visible browswer window size.
  • Viewport Height: vh | vh, or "viewport height", is a unit for scalable and responsive designs and is similar to vw. vh measures the total height of the browser window in the same way vw does.
