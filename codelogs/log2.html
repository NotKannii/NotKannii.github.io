"What is the difference between absolute and relative?"

What is absolute?

The absolute CSS tag is a unit of length. These lengths are usually fixed in a layout for one type of screen layout. For example, if a website is designed with absolute measurements, it will look normal on a computer, but small and unreadable on a phone.

What about relative?

The relative CSS tag is also a unit of length. However, unlike its absolute counterpart, relative measurements are responsive and adapt to the screen it is being shown on. It provides the best flexibiliy across different platforms and devices, making it readable no matter how big or small a screen is.
